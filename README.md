# Magnetic Soft Robot Closed Loop Control Framework: Model Inference via Reinforcement Learning and Multi-Layer Perceptron Ensemble

This repository contains the code and data used in the paper "Magnetic Soft Robot Closed Loop Control Framework: Model Inference via Reinforcement Learning and Multi-Layer Perceptron Ensemble" submitted to the IEEE Transactions on Robotics (TRO).
